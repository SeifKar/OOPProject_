# Shopping Cart

This is a simple shopping cart web application that allows users to browse products and add them to their cart. Users can also view the cart items, update the quantity of each product, and remove items from the cart. The cart calculates the total price and the total number of items in real time.

## Table of Contents
- [Project Overview](#project-overview)
- [Features](#features)
- [Technologies Used](#technologies-used)

## Project Overview

This project consists of an HTML page to display a list of products, which includes details like images, names, prices, and an "Add to Cart" button for each item. When a user clicks "Add to Cart," the item is added to the shopping cart, and the cart summary is displayed, showing the total number of items and the total price. Users can also remove items from the cart.

## Features

- Display a list of products with their images, names, and prices.
- Add products to the shopping cart.
- View cart details, including the name, quantity, and total price of each product.
- Remove products from the cart.
- Real-time update of the total number of items and total price in the cart.

## Technologies Used

- HTML5
- CSS3
- JavaScript (OOP)
- JSON for product data
